## Часть II. Приложение «Тёмная комната» — проект

### Концепция

**Не просто бот. Полноценное приложение, где Эриксон — твой личный секс-коуч.**

Пользователь общается с AI-клоном Эриксона: тот же стиль, тот же юмор, те же знания. Но бот ведёт пользователя по персональной программе, даёт упражнения, отслеживает прогресс, предлагает курсы в нужный момент.

### Telegram Mini App vs Native App

| Фактор | Telegram Mini App | Native App (iOS/Android) |
|--------|-------------------|-------------------------|
| Время разработки | 4–6 недель | 3–6 месяцев |
| Стоимость | 150K–300K ₽ | 1–3M ₽ |
| Доступ к аудитории | 41K уже в TG | Нужно скачать из стора |

| Платежи | Telegram Stars / YooKassa | Apple/Google берут 30% |
| Push-уведомления | Через TG | Нативные |

**Рекомендация: Telegram Mini App.** Аудитория уже в Telegram, модерация сторов не проблема, разработка ×5 дешевле.

## Часть III. Геймификация — детальный разбор

### Система прогрессии

| Механика | Как работает | Зачем |
|----------|-------------|-------|
| **Streaks** | Ежедневный вход + 1 действие = streak | Retention. Как Duolingo |
| **XP / Очки** | Урок = 50 XP, тренировка = 30 XP, квиз = 20 XP | Мотивация продолжать |
| **Уровни** | Новичок → Практик → Продвинутый → Мастер → Легенда | Статус, разблокировка контента |
| **Ачивки** | «7 дней подряд», «Все уроки модуля», «50 вопросов боту» | Дофаминовые петли |
| **Парные челленджи** | «Обсудите 3 фантазии», «Неделя прелюдий», «Новая поза каждый день» | Вовлечение партнёрши = retention ×2 |
| **Еженедельные квизы** | 5 вопросов на знания из пройденных уроков | Закрепление + контент для канала |

### Игровые механики для курсов

| Игра | Описание | Курс |
|------|----------|------|
| **«Анатомический атлас»** | Интерактивная схема — тапай по зонам, получай описание техник | Женский оргазм |
| **«Таймер мастера»** | Трекер длительности с советами в реальном времени | Продление акта |
| **«Рулетка фантазий»** | Парная игра — оба крутят, выпавшее пробуют | Все курсы |
| **«Дневник прогресса»** | Личный трекер: что пробовал, что сработало, рейтинг | Все курсы |
| **«Карта Кама Сутры»** | Визуальная карта поз — открываются по мере прохождения | Позиции |
| **«Challenge недели»** | Еженедельное задание от Эриксона с бонусными очками | Retention |

## Часть V. Мульти-экспертная оценка (10 ролей)

### 🧊 1. Скептик

> «Очередное приложение с ботом. Рынок завален. Почему кто-то будет платить 2К/мес за бота, если можно спросить ChatGPT бесплатно?»

**Ответ:** ChatGPT не знает контент Эриксона. Не ведёт по программе. Не даёт парные челленджи. Не трекает прогресс. Здесь ценность — персонализированная программа от конкретного эксперта, а не generic AI.

### 💰 2. Инвестор / финансист

> «Unit economics: CAC? LTV? Payback period?»

| Метрика | Значение |
|---------|----------|
| CAC (из TG канала) | ~0 ₽ (органика — 41K уже есть) |
| CAC (платный трафик) | 200–500 ₽ |
| LTV (Pro 6 мес) | 9 000 ₽ |
| LTV/CAC | 18–45x (органика) / 18–45x |
| Payback | < 1 месяца |

> «Цифры красивые. Но 41K — потолок? Откуда рост?»

Рост: YouTube (200K) → конвейер коротких видео → TG. Плюс виральность парных челленджей (девушка ставит парню, парень приводит друзей).

### 📱 3. Продуктовый менеджер

> «MVP — что, кому, когда?»

**MVP за 4 недели:**
1. Telegram Mini App
2. AI-чат (GPT + RAG на контенте)
3. Онбординг-квиз → персональный план
4. 1 бесплатный курс (прелюдия, 5 уроков)
5. Streak-система
6. Оплата через Telegram Stars

Остальное — итерация: парные челленджи (месяц 2), геймификация (месяц 3), полная библиотека (месяц 4).

### 🎨 4. UX/UI дизайнер

> «Тёмная тема — обязательно. Бренд "Тёмная комната" обязывает. Тёмный фон, акценты красным/оранжевым. Минимум текста, максимум визуала. Интерфейс как у premium-приложения, а не как инфобиз-бот на коленке.»

Референсы: Coral (чистый UI, парные экраны), Calm (dark mode, плавные анимации), Duolingo (геймификация).

### 📊 5. Маркетолог

> «Главный козырь — бесплатный AI-бот. Это лид-магнит №1. Человек задаёт вопрос → получает крутой ответ → видит "3 бесплатных вопроса сегодня, безлимит — в Pro". Конверсия гарантирована.»

**Воронка:**
```
Пост в канале: "Задай мне любой вопрос о сексе"
→ Ссылка на Mini App
→ Бот отвечает (бесплатно, 3 вопроса)
→ "Хочешь персональный план? Пройди квиз"
→ Квиз → план → бесплатный курс
→ "Нравится? Pro — 1990₽/мес"
```

Виральный цикл: парные челленджи → девушка делится → подруга спрашивает → новый пользователь.

### 🔒 6. Юрист

> «Риски: персональные данные о сексуальной жизни — это sensitive data по 152-ФЗ. Нужно: согласие на обработку, шифрование, серверы в РФ, возрастная верификация 18+.»

**Митигация:**
- Анонимные аккаунты (Telegram ID, без ФИО)
- Шифрование чат-истории (AES-256)
- Серверы в РФ (Selectel / Yandex Cloud)
- Age gate при первом входе
- Политика конфиденциальности с явным согласием

### 🏋️ 7. Сексолог / врач

> «Бот не должен ставить диагнозы. Если у мужчины стабильная ЭД — это к урологу, а не к боту. Бот должен чётко отделять "прокачку" от медицинских проблем.»

**Решение:** AI имеет жёсткие правила:
- При ключевых словах (кровь, боль, 3+ месяца проблем) → «Это к врачу. Вот чек-лист для визита к урологу.»
- Не рекомендует препараты
- Не ставит диагнозы
- Дисклеймер: «Это образовательный контент, не медицинская консультация»

### 🤖 8. AI-инженер

> «RAG на контенте Эриксона — правильный подход. Но качество ответов зависит от качества базы. Нужно: транскрибировать ВСЕ курсы, все Q&A, все посты. Минимум 500K токенов контента для хорошего RAG.»

**Стек:**
- GPT API (gpt-5.4-mini для чата, gpt-5.4 для сложных вопросов)
- Embeddings → Pinecone / Qdrant для RAG
- Whisper для транскрибации видеокурсов
- System prompt: стиль Эриксона + правила безопасности
- Fallback: «Не уверен — спроси Эриксона напрямую в VIP-чате»

### 💀 9. Пользователь-скептик (ЦА)

> «Нафиг мне бот, если я могу просто посмотреть видос на YouTube? Бесплатно.»

**Ответ через ценность:**
- YouTube не даёт персональный план
- YouTube не отслеживает прогресс
- YouTube не даёт парные упражнения
- YouTube не отвечает на ТВОЙ конкретный вопрос в 3 ночи
- YouTube не мотивирует streak'ами и ачивками

Бот = YouTube-контент + личный тренер + трекер + игра. Это другой продукт.

### 🏴‍☠️ 10. Пират / конкурент

> «Курсы уже сливают на slivinfo.ru. Что помешает слить приложение?»

**Защита:**
- AI-бот невозможно «слить» — он работает на сервере
- Персонализация привязана к аккаунту
- Streaks и ачивки нельзя скопировать
- Парные функции требуют двух аккаунтов
- Контент обновляется еженедельно — слив устаревает за месяц

**Главный аргумент:** слив статичного курса — это PDF. Слить живое приложение с AI — невозможно.
